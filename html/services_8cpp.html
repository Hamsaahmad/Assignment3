◆ startVM()

int startVM ( VM_Id_t  vmId)

Starts execution of a VM selected by the scheduler.

This function is called after the scheduler selects a VM that is in READY state. It updates the currently running VM in the hypervisor control block and performs the low-level context switch to transfer execution to that VM.

Parameters
vmIdVirtual machine ID
Returns
true if the VM was started successfully, false otherwise

Stops a virtual machine.
